## งานประจำวัน (ตามบทบาท)

### รีเซปชั่น — รับลูกค้า
1. ลูกค้าเดินเข้าร้าน → กดปุ่มม่วง **"รับ Walk-in"** → พิมพ์ชื่อ 2 ตัวอักษร เลือกลูกค้าเดิม (หรือพิมพ์ชื่อใหม่+เบอร์) → เลือกบริการ → **เข้าคิวเลย**
2. ลูกค้าโทรจอง → **"สร้างการจอง"** → เลือกวัน-เวลา / ลูกค้าจองมาถึง → การ์ดนัด → **เช็คอิน**
3. เรียกคิว → กดการ์ดคิว → เลือกหมอนวด (ระบบโชว์เฉพาะคนว่างที่ทำบริการนั้นได้) → **เรียกคิว**
4. เปิดจอ TV เรียกคิวให้ลูกค้าดู: ปุ่ม "จอเรียกคิว (TV)"

### หมอนวด — บนมือถือได้
เข้าระบบ → เห็นงานวันนี้ของตัวเอง (ป้ายส้ม ⚠ = ข้อควรระวังลูกค้า เช่นแพ้น้ำมัน) → กด **เริ่มงาน** ตอนรับลูกค้า → กด **จบงาน** เมื่อเสร็จ (หน้าร้านเห็นคิวรอชำระทันที) → ดูค่ามือเดือนนี้ + กดขอลาได้ (รอผู้จัดการอนุมัติ)

### แคชเชียร์ — เก็บเงิน
คิวที่จบงานขึ้น **"✅ จบงาน · รอชำระ" กรอบเขียวบนสุด** → กดโหลดเข้าบิล → ใส่คูปอง/ตัดคอร์ส (ถ้ามี) → เลือกช่องทาง:
- **เงินสด**: กดปุ่มลัดรับเงิน ระบบคำนวณทอนให้
- **QR พร้อมเพย์**: ระบบสร้าง Thai QR พร้อมยอด ลูกค้าสแกนด้วยแอปธนาคารใดก็ได้ (ตั้งเบอร์ PromptPay ร้านในหน้าตั้งค่า → field promptpayId)
→ **รับชำระ** → ใบเสร็จ/ใบกำกับภาษีขึ้นให้พิมพ์

### ผู้จัดการ/เจ้าของ — ดูภาพรวม
แดชบอร์ด: รายรับวันนี้/เดือน (กดการ์ดดูรายละเอียด), ตารางหมอนวดเลื่อนดูทั้งวัน, ปฏิทินจุดวันมีจอง / หน้า "พนักงาน": จัดกะ, อนุมัติลา, รีเซ็ตรหัส / หน้า "การเงิน": บิลย้อนหลัง+พิมพ์ซ้ำ, บันทึกรายจ่าย / หน้า "รายงาน": บริการขายดี, ค่ามือ, ช่วงเวลาคนเยอะ
